# Pinterest Clone
This is an example of the UI for Pinterest. 

## About this project
This project is based on the excellent book [Learning Pure React](https://daveceddia.com/learn-pure-react/) by Dave Ceddia

## Getting started
clone this repo and delete the .git file

### Download packages:
```
npm install
```

### Run the app:

```
npm start
```
